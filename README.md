# FrameScript

FrameScript é uma API para a geração de legendas embutidas em video. Ela usa da biblioteca Ffmpeg, a versão de modelo mais avançado do Whisper e processamento com sistema de filas e concorrência para melhorar a experiência de quem usa.

### Arquitetura
API **Estruturada em camadas** baseado em modulos, serviços e controladores comuns em projetos utilizando NestJS. Grande foco em modularização, **separação de responsabilidades** e reutilização de código com **programação orientada a objetos**.
